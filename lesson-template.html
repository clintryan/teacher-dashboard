to Data
Understanding what we collect and why it matters
Learning Objectives
  • Distinguish between categorical and numerical data
  • Identify discrete vs. continuous numerical variables
  • Give real-world examples of each data type
Concept

Two Kinds of Data

Categorical Data
Data that places items into named groups or categories. Arithmetic is not meaningful.
eye colour type of pet favourite subject postcode
Numerical Data
Data that represents measured or counted quantities. Arithmetic operations make sense.
Discrete
Counted — takes separate, distinct values (often whole numbers)
number of siblings goals scored shoe size
Continuous
Measured — can take any value in a range
height (cm) 100m time (s) rainfall (mm)
Key Idea
Ask: "Does adding or averaging these values make sense?"
Yes → numerical. No → categorical.
[ Diagram: Data Type Tree ]
Watch out: Numbers that label groups (jersey numbers, postcodes) are categorical — even though they look like numbers.
Worked Example

Classifying Variables

PROBLEM
For each variable, identify whether it is categorical, discrete numerical, or continuous numerical.
a. Shoe size (e.g. 7, 7.5, 8 …)
b. Eye colour
c. Time to run 100 m (seconds)
d. Number of texts sent today
Common Error
Students often call shoe size continuous because shoes feel like a measurement. But sizes come in fixed steps — you can't buy a size 8.37 — so it is discrete.
Solution
a
Discrete Numerical
Shoe sizes are fixed increments — counted, not continuously measured. Averages make sense.
b
Categorical
Eye colour places people into named groups. Averaging colours makes no sense.
c
Continuous Numerical
Time can take any value (12.37 s, 12.371 s…). It is measured, not counted.
d
Discrete Numerical
You count whole texts. The values are separate integers — you can't send 3.7 texts.

Reflection

True or False?

Statement T F Answer
Postcodes are numerical data because they are made up of digits. ☐ ☐ FALSE
The number of students in a classroom is an example of discrete data. ☐ ☐ TRUE
Height measured in centimetres is an example of continuous data. ☐ ☐ TRUE
Every variable is either numerical or it is useless for statistical analysis. ☐ ☐ FALSE
You can always compute the mean (average) of any numerical variable. ☐ ☐ TRUE
Reminder
Categorical data is extremely useful in statistics — we use frequency tables, bar charts, and mode to analyse it. Don't dismiss it as less important than numerical data.

Prior Knowledge Required
  • Students should know what a "variable" means in context
  • Basic familiarity with real numbers and decimals
Anticipated Misconceptions
  • Postcodes as numerical: Most students will say postcodes are numerical. Emphasise the digits don't represent a quantity — you wouldn't add or average them.
  • Shoe size as continuous: Shoe sizes feel like a measurement but come in fixed increments. Use this to drive home the discrete/continuous distinction.
  • Discrete = integers only: Technically discrete means countable distinct values; shoe sizes (7, 7.5, 8…) are discrete despite including halves.
Extension — Ordinal Data
  • Ask fast finishers: "Is a movie rating (1–5 stars) categorical or numerical?"
  • Introduce ordinal data — categories with a meaningful order but unequal gaps (e.g. Likert scales). Bridge to senior statistics.
  • Challenge: find a real dataset online and classify every variable in it.
