Motivation

Why Does Standard Deviation Matter?

Two companies. Same average salary of ฿45,000/month. Which would you rather work for?

Company A — Wide Spread

σ ≈ ฿22,000 — Salaries range from ฿10k to ฿80k

Company B — Tight Cluster

σ ≈ ฿3,000 — Nearly everyone earns close to ฿45k

"Both groups earn the same on average. Are they equally well-off?"
Teacher Note Company A has extreme inequality — a junior worker earns ฿10k while a manager earns ฿80k. Company B is far more equitable. The mean hides this entirely. Standard deviation is the story behind the number.
Vocabulary & Notation

Key Terms — Chapter 2

Mean
\(\bar{x}\)  "x-bar"
Pronunciation: "x bar"
The sum of all values divided by the total frequency. The arithmetic average. Can be affected by extreme values (outliers).
Median
\(Q_2\) or \(\tilde{x}\)
Pronunciation: "mee-dee-an"
The middle value when data is ordered. Splits the distribution 50/50. Resistant to outliers — a better measure for skewed data.
Variance
\(\sigma^2\)  "sigma squared"
Pronunciation: "sig-ma"
The mean squared deviation from the mean: \(\sigma^2 = \frac{\Sigma fx^2}{\Sigma f} - \bar{x}^2\). Measures spread in squared units.
Std Dev
\(\sigma\)  "sigma"
Pronunciation: "sig-ma"
The square root of variance: \(\sigma = \sqrt{\sigma^2}\). Returns to the original units of measurement. Most commonly quoted measure of spread.
Teacher Note Emphasise that \(\sigma^2\) and \(\sigma\) are population parameters (lowercase Greek). Sample equivalents use \(s^2\) and \(s\) — but S1 uses population formula throughout. Students frequently confuse these.

Conditional Probability — Think It Through

A bag contains 3 red and 2 blue balls. Answer each question before moving on.

1
You draw one ball. What is \(P(\text{Red})\)?
2
You keep the first ball (it was red). What is \(P(\text{Red})\) on the second draw?
3
Instead, you replace the first ball before drawing again. What is \(P(\text{Red})\) on the second draw?
4
Why does the calculation change between questions 2 and 3?
Teacher Answers
Q1 Answer\(\frac{3}{5} = 0.6\)
Q2 Answer\(\frac{2}{4} = \frac{1}{2}\) (one red removed, 4 remain)

Worked Example

Mean & Standard Deviation from a Frequency Table

PROBLEM

Times (minutes) taken by 20 students to complete a puzzle:

Time \(t\) (min)Frequency \(f\)
\(5 \le t < 10\)3
\(10 \le t < 15\)8
\(15 \le t < 20\)6
\(20 \le t < 25\)3

Find: (a) the mean   (b) the standard deviation

Solution — click to reveal each step

Step 1 — Use class midpoints:
\(x =\) 7.5, 12.5, 17.5, 22.5
Step 2 — Check \(\Sigma f\):
\(\Sigma f = 3 + 8 + 6 + 3 = 20 \checkmark\)
Step 3 — Calculate \(\Sigma fx\):
\(\Sigma fx = 3(7.5) + 8(12.5) + 6(17.5) + 3(22.5)\)
\(= 22.5 + 100 + 105 + 67.5 = 295\)
Step 4 — Find the mean:
\(\bar{x} = \dfrac{\Sigma fx}{\Sigma f} = \dfrac{295}{20} = \mathbf{14.75} \text{ minutes}\)
Step 5 — Calculate \(\Sigma fx^2\):
\(\Sigma fx^2 = 3(56.25) + 8(156.25) + 6(306.25) + 3(506.25)\)
\(= 168.75 + 1250 + 1837.5 + 1518.75 = \mathbf{4775}\)
Step 6 — Find the Variance:
\(\sigma^2 = \dfrac{\Sigma fx^2}{\Sigma f} - \bar{x}^2 = \dfrac{4775}{20} - 14.75^2\)
\(= 238.75 - 217.5625 = \mathbf{21.1875}\)
Step 7 — Final Answer:
\(\sigma = \sqrt{21.1875} \approx \mathbf{4.60 \text{ minutes}} \quad (3 \text{ s.f.}) \checkmark\)
Spot the Error

Find and Fix the Mistake

A student attempted to find the mean from the same frequency table. Find the error and correct it.

Line 1:  \(\bar{x} = \dfrac{3 + 8 + 6 + 3}{4} = \dfrac{20}{4} = 5\)

Line 2:  Midpoints: 7.5, 12.5, 17.5, 22.5

Line 3:  \(\Sigma fx = 3(7.5) + 8(12.5) + 6(17.5) + 3(22.5) = 295\)

Line 4:  \(\Sigma fx^2 = 3(56.25) + 8(156.25) + 6(306.25) + 3(506.25) = 4775\)

Line 5:  \(\sigma^2 = \dfrac{4775}{20} - 5^2 = 238.75 - 25 = 213.75\)

Line 6:  \(\sigma = \sqrt{213.75} \approx 14.62\)  ✗

Where is the error? What should lines 1 and 5 say?

Teacher Note — Error Location & Correction Error is in Line 1. The student divided the sum of frequencies by the number of classes (4), not by \(\Sigma f\) (20). The correct formula is \(\bar{x} = \dfrac{\Sigma fx}{\Sigma f} = \dfrac{295}{20} = 14.75\). This cascades into Line 5 where \(\bar{x}^2\) should be \(14.75^2 = 217.5625\), giving variance = 21.1875 and \(\sigma \approx 4.60\).

Question 1
What happens to the normal curve as \(\sigma\) increases? Describe in terms of height and width.
Answer Curve gets shorter and wider — total area remains 1. As \(\sigma \to 0\), curve becomes a spike; as \(\sigma \to \infty\), it becomes completely flat.
Question 2
If \(\mu = 0\) and \(\sigma = 1\), what is \(P(X < 0)\)? Why?
Answer \(P(X < 0) = 0.5\). The normal distribution is perfectly symmetric about \(\mu\), so exactly half the distribution lies below the mean.
Chapter Summary

Key Formulas — Chapter 2

Core Formulas

1 \(\bar{x} = \dfrac{\Sigma fx}{\Sigma f}\)
2 \(\sigma^2 = \dfrac{\Sigma fx^2}{\Sigma f} - \bar{x}^2\)
3 \(\sigma = \sqrt{\sigma^2}\)
4 Coded: if \(y = \dfrac{x-a}{b}\) then \(\bar{y} = \dfrac{\bar{x}-a}{b}\) and \(\sigma_y = \dfrac{\sigma_x}{b}\)

Common Pitfalls

Dividing by the number of classes instead of \(\Sigma f\) when computing \(\bar{x}\).
Forgetting to square \(\bar{x}\) when computing variance (computing \(\frac{\Sigma fx^2}{\Sigma f} - \bar{x}\) instead of \(-\bar{x}^2\)).
Using raw data values instead of midpoints for grouped frequency tables.
Exam Connection: These formulas appear on S1 Papers 1–2 (1–2 marks each). The coded data formula is particularly common in Paper 1 "show that" questions.  — Also remind students the formula sheet is provided; marks come from correct application, not memory.
Extension Challenge

Variance of a Discrete Uniform Distribution

A fair six-sided die is rolled. Let \(X\) be the score shown.

Without using a calculator, prove that \(\text{Var}(X) = \dfrac{35}{12}\).

Hint: Start with \(E(X) = \dfrac{7}{2}\) (the mean of 1,2,3,4,5,6). Then find \(E(X^2) = \displaystyle\sum_{x=1}^{6} x^2 \cdot \tfrac{1}{6}\). Finally apply \(\text{Var}(X) = E(X^2) - [E(X)]^2\).
Full Solution (Teacher Only)
\(E(X) = \frac{1+2+3+4+5+6}{6} = \frac{21}{6} = \frac{7}{2}\)
\(E(X^2) = \frac{1+4+9+16+25+36}{6} = \frac{91}{6}\)
\(\text{Var}(X) = \frac{91}{6} - \left(\frac{7}{2}\right)^2 = \frac{91}{6} - \frac{49}{4} = \frac{182}{12} - \frac{147}{12} = \frac{35}{12} \checkmark\)

Connecting Ideas

From Discrete to Continuous Distributions

You Already Know…

Discrete Distributions
e.g. fair die, biased coin
Each value \(x\) has exact probability \(P(X = x)\)
\(\displaystyle\sum_{\text{all } x} P(X = x) = 1\)
Displayed as a bar chart — bars have definite width and height
\(E(X) = \Sigma x \cdot P(X=x)\)

Now We Extend To…

Continuous Distributions
e.g. heights, times, weights
\(P(X = x) = 0\) for any exact value — there are infinitely many!
\(\displaystyle\int_{-\infty}^{+\infty} f(x)\, dx = 1\)
Displayed as a smooth curve — area under curve = probability
\(P(a \le X \le b) = \displaystyle\int_a^b f(x)\, dx\)
"What must be true about the total area under any probability distribution curve?"
Teacher Note Answer: Total area = 1 (certainty). This bridges directly to Chapter 7 (Normal Distribution). The Normal curve has \(\mu\) at its peak and \(\sigma\) controlling spread — both concepts from Chapter 2 now reappear in a continuous context.
